# Action

The smallest unit of interaction with a computer interface.

## Components

- **type**: What to do (CLICK, TYPE, SCROLL)
- **argument**: Additional data (coordinates, text)
- **count**: Repetitions (optional)

## Example

```json
{
  "type": "CLICK",
  "argument": "500,300"
}
```
